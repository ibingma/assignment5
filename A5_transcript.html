Assignment 5: Demonstrate Use of JavaScript

The first task was to change the background color of the homepage at regular intervals. These color changes are applied to both the HTML and body elements' backgrounds. The intervals are set at 5 seconds.

Adding to the festive atmosphere, a countdown timer is set—not for a specific birthday but rather for New Year's Day 2025, building anticipation and aligning with the celebratory tone of the page. Between the navigation bar and the footer, a silhouette of people dancing serves as a dynamic background image, bridging the sections and further enhancing the lively feel.

These are the JavaScript codes, where I used a function called changeBackgroundColor to cycle through different colors. These colors, which are specific to the homepage, are stored in an array. Both the colors array and the colorIndex variables are defined outside the function so that they don’t have to be redefined with each interval. This is especially important for colorIndex because it needs to increment and remember its position every time the function is called.

The colorIndex cycles through the array continuously without exceeding its bounds. This is achieved using the modulo operator (%), which resets colorIndex to the beginning of the array once it reaches the end, creating a smooth loop through the colors.

For the backdrop, I chose a stunning sunset photo taken from one of Bali’s beautiful beaches, adding a serene and visually striking element.

The navigation bar has been repositioned to sit at the bottom of the image, providing easy access without distracting from the main content.

Aside from the timer, there’s a single paragraph to keep the page simple and uncluttered.

JavaScript code powers the countdown timer functionality. The countdown function itself is adapted from a tutorial in the textbook, with modifications for flexibility and style.

I moved the targetDate variable outside the function for easy access, allowing quick adjustments to the countdown date.

Additionally, I included if-else statements to change the color of the timer and text as the target date approaches, adding a visual cue as the event draws nearer.
